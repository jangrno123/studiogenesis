## GitHub Pages 배포

이 프로젝트는 `main` 브랜치에 변경 사항이 올라오면 GitHub Actions를 통해 자동 배포됩니다.

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 로컬 저장소에 GitHub 원격 주소를 연결합니다.
3. `main` 브랜치를 GitHub에 올립니다.
4. GitHub 저장소의 **Settings → Pages → Build and deployment**에서 Source를 **GitHub Actions**로 지정합니다.

배포가 완료되면 다음 형태의 공개 주소로 누구에게나 전달할 수 있습니다.

`https://사용자명.github.io/저장소명/`
